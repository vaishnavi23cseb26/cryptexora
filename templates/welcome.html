Welcome to Cryptexora IDS

Cryptexora is a web-based Intrusion Detection System that analyzes network traffic in real-time using simple, transparent rule-based logic. No AI black-boxes — just clear, understandable detection you can learn from.

🔍 Start Detection 📋 View Logs
🔍

Traffic Analysis

Analyze IP addresses, packet sizes, and request types with instant results. Understand exactly why a packet is flagged.

⚡

Rule-Based Detection

Uses simple if-else logic rules that are easy to read and extend. Perfect for learning how real IDS systems work.

📋

Audit Logs

Every scan is automatically saved with timestamp, IP, result, and threat details. Full traceability of all activity.

📊

Live Dashboard

See totals for scans, attacks detected, and normal traffic at a glance. Monitor your security posture in real-time.

🗺️ How Cryptexora Works

1

Input Traffic Data

Enter an IP address, packet size in bytes, and the HTTP request type you want to analyze.

2

Rule Engine Runs

Cryptexora checks each value against its detection rules — suspicious IPs, large packets, dangerous methods.

3

Get Instant Verdict

Receive a clear NORMAL or ATTACK result with severity level and a plain-English explanation of findings.

4

Log Everything

Each result is saved automatically. Visit the Logs page to review history and export findings.
